# Sympolog Sticky Behavior Implementation Plan

シンポログ（右下の吹き出しアイコン・FAB）をドラッグして画面右端に移動した際、右端に吸着し、縦長の形状に変化する機能を実装します。

## Proposed Changes

### 1. `style.css`
- **[MODIFY]** `style.css`
  - 新たなクラス `.fab-container.sticky-right` を追加します。
  - このクラスが付与された際、FABが縦長（例：幅を狭くし、高さを広げる、またはアイコンの配置を変えるなど）になるようスタイルを定義します。
  - ボーダーラジアスの調整や、右端にピッタリくっつくような余白（margin/right）の調整を行います。
  - スムーズな変形のために `transition` を追加します。

### 2. `script.js`
- **[MODIFY]** `script.js`
  - `fab-container` のドラッグ（`touchmove`, `mousemove` または専用のドラッグ関数）終了時、または移動中の座標を監視します。
  - X座標が画面の右端付近（例：右端から50px以内など）に達した場合、強制的に右端（`right: 0`付近）に座標をスナップ（固定）させます。
  - スナップした際に `sticky-right` クラスを付与し、縦長に変形させます。
  - 左側（または中央側）に向かって再度ドラッグされた場合は、`sticky-right` クラスを外し、元の丸い形状に戻します。

## Verification Plan
### Manual Verification
- ローカルサーバーでサイトを開き、シンポログアイコンをドラッグする。
- 画面右端に持っていった際に、カチッと右端に吸着し、形が縦長になるか確認する。
- 縦長になった状態から再度左へドラッグすると、元の丸い形に戻り、自由に動かせるようになるか確認する。
